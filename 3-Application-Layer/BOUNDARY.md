# 3-Application-Layer — Boundary

## Purpose

This folder will contain application-layer components that surface
domain truth to end users and external systems.

## Integrity Level

Level 3 — Application Layer

## Rules

- Application code must not own truth — it surfaces truth from lower layers
- UI components, API surfaces, and user-facing workflows belong here
- All application outputs are derived artifacts, not truth owners
- Modifications require L1 or L2 authorization
